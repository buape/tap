# Buape Tap

## How do I install these formulae?

`brew install buape/tap/<formula>`

Or `brew tap buape/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "buape/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
